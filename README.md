## ✦ About

I'm a Full Stack Developer and AI enthusiast based in India, building at the intersection of modern web technologies and artificial intelligence. My approach to engineering is rooted in minimalism and performance—every line of code should be intentional, and every architecture decision should serve the end user.

Currently, I'm working with **Next.js**, **TypeScript**, and **Python**, exploring advanced AI/ML architectures and building scalable systems.
